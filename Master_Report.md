## 2. Project Summary
The Chitkaar Welfare Society Platform is a state-of-the-art web application designed to streamline NGO operations. It leverages a modern tech stack (Next.js, Firebase, Contentful) to deliver a seamless experience for volunteers and administrators alike.

## 3. Key Deliverables
*   **Source Code:** Fully functional Next.js application.
*   **Documentation:** 15 Comprehensive Lab Reports covering every phase of the SDLC.
*   **Live Deployment:** Hosted on Vercel ensuring 99.9% uptime.
